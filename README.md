# College Placement Data Explorer

## Problem Statement
College placements are often discussed using assumptions and anecdotes.
This project aims to analyze placement data to understand which factors
most influence placement outcomes and salary offers.

## Objectives
- Explore trends in college placement data
- Identify key factors affecting placements and salaries
- Build simple machine learning models to predict outcomes

## Dataset
The dataset contains anonymized student placement data including
academic performance, skills, internships, and placement results.
